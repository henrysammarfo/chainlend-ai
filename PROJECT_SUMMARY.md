## 🚀 Getting Started

### Quick Setup
```bash
git clone https://github.com/yourusername/chainlend-ai.git
cd chainlend-ai
npm install
npm run dev
```

### Environment Configuration
```bash
cp env.example .env
# Add your ZetaChain RPC URL and private key
```

### Deploy Contracts
```bash
npm run compile
npm run deploy:testnet
```
